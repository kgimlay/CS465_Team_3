Detailed Description

JoinMessage is the subtype of Message that is sent from a node that wants to join the chat, to a known node in the chat topology. This is the first message a joining node sends. When sent by the joining node, the participant list is left Null. The node that is contacted sends a "join" message back in response, and fills the participant list with the list of participants to add to the joining node's participant list.
