Detailed Description

ReceiveThread handles receiving a message from a single connection in the topology. A ReceiveThread is spawned on the acceptance of an incomming connection request, it reads in the incomming message, and reacts to that message however is appropriate: ChatMessage - print to print stream (console) JoinMessage - responds with a Participant list JoinedMessage - adds the sending node to the Participant list LeaveMessage - removes the node from the Participant list. The thread terminates once the sending side of the connection closes the connection.
